[**顶点云设备管理系统**](http://zenith-monitor.forec.cn) 是我 2016 年的一项课程设计作业，最初是针对模拟设备的监控、管理，现在已支持对实体设备的监控和操纵，以及实时摄像和设备画面。顶点云设备管理系统的结构和实现非常简单，可运行在包括树莓派在内的任何计算机上，目前我将其部署在我的 CVM 上。

此仓库托管了顶点云设备管理系统的源码，你可以通过任何设备的浏览器访问顶点云设备管理系统。因为私人签署的证书不被浏览器信任，所以顶点云设备管理系统部署时放弃使用 https 协议。在公网部署的顶点云设备管理系统仅作展示用，不提供开放注册或任何实质功能，但你可以使用测试用户登录体验。
